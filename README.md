# 星穹抽卡链接获取工具

> 一键获取《崩坏：星穹铁道》抽卡记录分析链接，启动即用。

## ✨ 功能

- 自动检测分析 国服 / 海外服 游戏日志，获取最新跃迁链接
- 历史操作记录存档
- 适配 浅色 / 深色模式，支持自适应调节
- 完全本地化

## 🚀 使用

1. 打开《崩坏：星穹铁道》，进入跃迁页面，点击“查看详情”
2. 运行 `StarRail_WarpLinkTool`，选择合适的游戏服务器后，点击按钮获取
3. 链接会自动复制到剪贴板
4. 粘贴到任意抽卡分析工具使用（例如“小黑盒”）

## 📥 下载

从 [Releases页](https://github.com/LinineTy/StarRail-WarpLinkTool/releases) 下载最新的 `StarRail_WarpLinkTool`，然后双击即可运行。
